# WP Dummy Content Filler

WP Dummy Content Filler is a powerful WordPress plugin designed for QA testers and developers to generate realistic dummy posts, users, and WooCommerce products. It intelligently detects and populates custom meta fields from ACF, CMB2, and other frameworks with 20+ data types. Watch real-time progress with visual tracking and enjoy safe one-click cleanup of all dummy content. Perfect for testing themes, plugins, and site performance without manual data entry.

## Features
- Create dummy posts, users & WooCommerce products in bulk
- Automatically detect & populate ACF, CMB2 & custom meta fields
- Generate 20+ data types including names, emails, prices & dates
- Watch real-time progress with visual circular progress bar
- Create & assign taxonomies to posts and products automatically
- Safe one-click cleanup – deletes only dummy content, never real data

## Installation

### Method: WordPress Admin
1. Go to Plugins → Add New
2. Search for "WP Search Any"
3. Click Install Now
4. Click Activate
